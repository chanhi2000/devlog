### 사용할 수 있는 APIs

- `forest.http`: [산림청](http://sansatai.forest.go.kr)
- `hrfco.http`: [한국홍수통제소](http://api.hrfco.go.kr)
- `iermn.http`: [국가환경방사선자동감지망](https://iernet.kins.re.kr/IERNet.asmx/jsonData)
- `kakao.http`: [카카오](https://dapi.kakao.com/v2)
- `lics.http`: [가로등](http://211.231.15.66:40000/ilcs1/1280)
- `nsgii.http`: [국토교통부](http://map.ngii.go.kr/openapi)
- `seoul.http`: [서울 열린데이터광장](http://openAPI.seoul.go.kr:8088)
- `vworld.http`: [VWorld](http://api.vworld.kr/req)

### 🔑API Key 관리

사용하고 싶은 API의 키를 발급 받아 <FontIcon icon="iconfont icon-json"/>`settings.json` 파일을 만들어 환경설정 하세요.

```json
"rest-client.certificates": {
  "local": {
    "forest.key": "" /* 산림청 */,
    "hrfco.key": "" /* 한국홍수통제소 */,
    "iermn.key": "" /* 국가환경방사선자동감지망 */,
    "kakao.key": "" /* 카카오 */,
    "lics.key": "" /* 가로등 */,
    "nsgii.key": "" /* 국토교통부 */,
    "nsgii.rfrnUrl": "" /*  */,
    "seoul.key": "" /* 서울 열린데이터광장 */,
    "vworld.key": "" /* VWorld */,
  }   
}
```
