<p>iPhone X did away with the home button for the first time in any iPhone, replacing it with an interactive line that sits at the bottom of the screen called the home indicator.</p>
<p>This indicator always remains visible unless you request others, and generally Apple suggests it’s a good idea to keep it visible. At the same time, there are many times when you want full control of the screen to yourself, for example if the user is watching a video, and in those times you’re going to want to hide the home indicator.</p>
<p>To hide the home indicator on iPhone X all you need to do is return true from the <code>prefersHomeIndicatorAutoHidden</code> property in your view controller. You can add logic if you want, allowing you to hide the home indicator only when certain conditions are reached, or you can just always request it be hidden, like this:</p>
<pre class=" language-swift"><code class=" language-swift"><span class="token keyword">override</span> <span class="token keyword">var</span> prefersHomeIndicatorAutoHidden<span class="token punctuation">:</span> <span class="token class-name">Bool</span> <span class="token punctuation">{</span>
    <span class="token keyword">return</span> <span class="token boolean">true</span>
<span class="token punctuation">}</span></code></pre>
<p>That will cause the home indicator to be hidden after a couple of seconds, but it will reappear as soon as the user touches the screen.</p>
