<p>Closures and functions are very similar in Swift, but there are some subtle differences. When you’re learning, it’s easiest to think of a closure as being a function that doesn’t have a name of its own, and captures any values from its environment.</p>
<p>Nested functions – functions inside other functions – also capture values from their environment, but they have a name.
