## 유용한 커맨드

```sh
# 기동을 위한 준비
wscfl -i http.m   # http.m 컴파일
wsboot            # WebtoB 기동


# 서버운영 상태 확인
wsadmin           # wsadmin 접속
```

## `wsadmin` 안 실행어

```sh
si          # 서버 운영 상태 확인
```
